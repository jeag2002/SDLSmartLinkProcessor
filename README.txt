Ford/Toyota Android App SDL smartlink Project.

InApp android application; remote control to a ford/toyota car control; with the following features.

1)Connect to a remote SDL smartlink panel (emulated using Manticore)
2)Create embedded app
3)Capture RPM (Revolutions per minute) data and shows them in an android app.
4)Send RPM data to remote subscriber; using MQTT/Mosquitto.

Technologies used:

Windows 10 x64 professional edition
Java 1.8.0_181 SDK
Android Studio (Android v28 Oreo support)
FitChart:1.0
smartdevicelink:4.+
eclipse paho mqttv3:1.0.2
eclipse paho 1.0.2

Android chart plugin: txusballesteros FitChart 1.0
